Списки

Список определений

<dl> ― Definitions List;
<dt> ― Description Term;
<dd> ― Description Details;
Нумерованный список
― предназначен для перечисления элементов в точном порядке;
Маркированный список
― предназначен для перечисления элементов в произвольном порядке;
Список определений
― предназначен для перечисления терминов, и разъяснения этих терминов в виде определений;

Списки поддерживают вложенность, НО список НЕЛЬЗЯ вкладывать непосредственно в другой список, а можно вкладывать лишь в элемент другого списка.

Нумерованный список:

<ol> ― Ordered List;
<li> ― List Item (Элемент списка);
  1. Элемент 1;
  2. Элемент 2;
  3. Элемент 3;
  4. Элемент 4;
  5. Элемент 5;

Нумерованный список можно прервать, и продолжить позже.

  1. Элемент 11
  2. Элемент 12
  3. Элемент 13
  4. Элемент 14
  5. Элемент 15

Маркированный список

<ul> ― Unordered List;
<li> ― List Item (Элемент списка);
  • Элемент 1
  • Элемент 2
  • Элемент 3
  • Элемент 4
  • Элемент 5
