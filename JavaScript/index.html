JavaScript

Введение

JavaScript ― это интерпретируемый язык программирования общего назначения с динамической типизацией данных.

Язык программирования общего назначения означает, что на этом языке можно решать практически любые задачи по программированию.

Интерпретируемый означает то, что программы, написанные на этом языке выполняются в исхном виде, т.е., исходный код НЕ преобразуется в машинный или промежуточный код. Выполняется именно исходный код. Интерпретатор ― это программа, которая выполняет другие программы. Недостатком интерпретируемых языков является низкая производительность, а преимуществом интерпретируемых языков является высокая кроссплатформенность. Программа будет работать везде, где работает интерпретатор. Это дает возможность использовать JavaScript для разработки серверных приложений как под Windows, так и под UNIX, разработки мобильных приложений для iPhone/Android.... Web-разработка: Front-End Client-side; Back-End Server-side;

То что программа никуда не компилируется, а выполняется сам исходный код этой программы, говорит о том, что нет ошибок ни компиляции и компоновки, а есть только ошибки выполнения, что часто усложняет поиск ошибок.

Динамическая типизация данных означает то, что у переменной нет конкретного типа, ее тип зависит от того, какое в ней хранится значение. var a = 2; a = 3; a = "Вася"; (object) Upcast - приведение Дочернего значения к Базовому типу.

Переменные и консанты

Переменную можно объявить при помощи ключевого слова var или let.

var
объявляет глобальную переменную, независимо от того, в какой области видимости объявляется переменная. Т.е., переменная, созданная при помощи кюлчевого слова var будет доступна везде после объявления;
let
объявляет локальную переменную. Переменная, созданная при помощи let существует только в той области видимости, в которой она объявлена, и удаляется из памяти при выходе из своей области видимости;
const
объявляет константу. Константы являются исключительно локальными;

Если использовать несуществующую переменную, то это НЕ вызывает никаких ошибок, а просто неявно создает глобальную переменную.
